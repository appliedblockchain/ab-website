Cobalt

Cobalt is a fast, light-weight Ethereum development and testing library that saves developers valuable time when modifying, testing and deploying their dApps.

Cobalt uses instant compilation, the same APIs in development and production code as well as straight-forward deployment without migrations.

It is available on open-source, so go and try it out for yourself.
